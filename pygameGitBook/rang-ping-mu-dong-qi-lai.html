让屏幕动起来

让屏幕动起来

若要让屏幕循环动起来，需要加载两次背景图，第一张背景图放在 y0,x0的位置，第二张图片放在y-768，x0的位置，你可以想象第二个张图片叠在第一张图片的头上。

接下来，如果图片1 向下移动，那么图片2也向下移动，另外，当图片1超出屏幕的底部的时候，就要让它回到初始的位置(0,0)，同样，图片2到达屏幕的底部的时候，也让它回到初始位置，也就是 y-768,x0

  • 创建两张图片
# 背景图
bg1 = pygame.image.load('images/bg.png')
bg2 = pygame.image.load('images/bg.png')
  • 获取他们的矩形，并且要更改图片2的矩形
bg1_rect = bg1.get_rect()    # 获取图片1的矩形
bg2_rect = bg2.get_rect()    # 获取图片2的矩形
# 更改背景矩形2的位置为y-768，x0
bg2_rect.x = 0
bg2_rect.y = -768
  • 绘制到屏幕
screen.blit(bg1, bg1_rect)    # 绘制背景1
screen.blit(bg2, bg2_rect)    # 绘制背景2

移动背景图的矩形

  • 在循环体内移动两个背景图的矩形
# 移动矩形
bg1_rect.y += 1
bg2_rect.y += 1

屏幕是移动了，但是会出现一些奇奇怪怪的画面，原因是没有让它复原

背景图矩形复原

  • 在循环体中写：
# 复原
# 如果矩形1的y轴大于768的时候:
if bg1_rect.y > 768:
# 把矩形1的y轴赋值变成0
    bg1_rect.y = 0
if bg2_rect.y > 0:
    bg2_rect.y = -768
